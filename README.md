# Global Seismic Activity Analysis

A geospatial and statistical analysis project focused on understanding global earthquake behavior using seismic visualization, temporal analysis, and tectonic activity mapping.

---

## Project Overview

This project analyzes worldwide earthquake activity using the USGS Earthquake Dataset (Magnitude 2.5+ events). The analysis explores spatial distributions, seismic energy release, depth-magnitude relationships, temporal activity patterns, and monitoring limitations.

The project combines geospatial visualization, statistical analysis, and interactive dashboards to better understand seismic risk and tectonic behavior.

---

## Objectives

- Analyze global earthquake occurrence patterns
- Study relationships between earthquake depth and magnitude
- Visualize seismic energy distribution across tectonic regions
- Identify seismic hotspots and tectonic boundaries
- Explore temporal earthquake trends and clustering
- Examine seismic station coverage and detection completeness

Shows yearly earthquake counts, strongest events, and temporal patterns.

### Global Earthquake Occurrence Map
Visualizes tectonic plate boundary activity and seismic hotspots.

---

## Key Insights

- Most earthquakes occur along tectonic plate boundaries.
- High-magnitude earthquakes are concentrated at shallow to intermediate depths.
- The Pacific Ring of Fire remains the most seismically active region.
- Deep earthquakes are generally lower in magnitude.
- Subduction zones release the highest seismic energy globally.
- Detection limitations increase significantly with earthquake depth.
