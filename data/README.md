# Dataset Information

## USGS Earthquake Dataset

Source:
https://earthquake.usgs.gov/

Dataset:
Magnitude 2.5+ Earthquake Events

The dataset includes:
- Earthquake magnitude
- Depth
- Latitude and longitude
- Event timestamps
- Azimuthal gap measurements
- Seismic metadata

---

## Purpose of the Dataset

This dataset was used to:
- Analyze seismic activity patterns
- Study tectonic behavior
- Visualize earthquake distributions
- Explore depth-magnitude relationships
- Measure seismic energy release
- Investigate earthquake monitoring coverage
